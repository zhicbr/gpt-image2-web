# 图像编辑扩展规划

## 当前方向

项目当前是单图工作流，不做聊天式历史链路。  
当前图就是当前状态，后续参考图、继续编辑、局部修改都围绕这张图展开。

## 功能理解

### 参考图

- 支持上传参考图
- 未出图时，主图片区的空白方框同时作为上传入口
- 上传后，这张图作为输入参考图参与生成

### 继续编辑

- 生成结果图后，当前图可以直接作为下一轮参考图
- 不额外保留历史链
- 不增加独立历史面板
- 如果用户要保留旧图，自行下载

建议交互：

- 图片右上角显示 3 个图标：
  - 详细信息
  - 编辑
  - 下载
- 点击“编辑”后：
  - 当前图片自动进入参考图状态
  - 左侧继续使用现有提示词输入区填写修改要求
  - 生成后直接替换当前图片

### 局部修改

- 局部修改不同于单纯用文字说明“改哪里”
- 文字说明约束弱，模型可能改动其他区域
- 局部修改更适合使用 `mask`
- 即通过涂抹/圈选指定区域，只重绘该部分

## 参数展示

- 输出参数后续可以补充支持
- 为避免破坏主界面，建议放在“详细信息”面板中
- 不直接长期展示在主画面

建议展示内容：

- format
- size
- quality
- background
- compression
- 是否使用参考图
- revised prompt

## 优先级

1. 详细信息面板
2. 参考图上传
3. 当前图片继续编辑
4. 局部修改（mask）
5. 流式中间图预览

## 说明

- 流式中间图预览不是当前优先项
- 第一阶段先做整图继续编辑
- 第二阶段再考虑局部精修
