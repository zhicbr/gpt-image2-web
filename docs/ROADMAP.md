# 产品方向

当前版本基线为 `0.1.0`。

## 功能划分

- 直接功能：接口直接提供的能力，如生图、编辑图。
- 通用功能：UI、提示词预设、历史记录、错误处理、加载态、结果展示等。

## 当前状态

- 当前只保留一个直接功能：`生图`
- `生图` 已经跑通

## 当前方向

当前阶段不增加新的直接功能。  
接下来只做通用功能优化：

- UI 重写
- 提示词预设
- 其余通用体验优化

前端后续改为 `React + Vite`，后端先保持不变。

## 版本规则

如果后续新增底层直接功能，主版本号加 `1`。  
仅做 UI 和通用功能优化，不算新增直接功能。
