# 命名规范

本文档用于后续讨论、改版和提需求时，统一页面区域与功能模块的叫法。  
本文档不是组件命名规范，不约束具体实现方式。

## 页面一级区域

整个页面分为以下一级区域：

- 提示词工作区
- 结果展示区
- 状态反馈区

说明：

- 提示词工作区：负责输入、调整和发起生成
- 结果展示区：负责展示生成结果及结果操作
- 状态反馈区：负责显示当前状态、错误、空状态、加载状态等

## 提示词工作区

提示词工作区内部统一使用以下叫法：

- 提示词输入区
- 参数选择区
- 预设提示词区
- 生成操作区

说明：

- 提示词输入区：用户写提示词的主要输入位置
- 参数选择区：尺寸、格式、质量等可选项所在区域
- 预设提示词区：后续放提示词预设、分类、快捷填充
- 生成操作区：生成按钮及其直接相关操作

## 结果展示区

结果展示区内部统一使用以下叫法：

- 结果列表区
- 单张结果卡
- 结果说明区
- 结果操作区

说明：

- 结果列表区：用于承载全部生成结果
- 单张结果卡：单张图片及其附属信息
- 结果说明区：提示词修订、生成说明等文本信息
- 结果操作区：下载、查看、复制等操作

## 状态反馈区

状态反馈区统一使用以下状态叫法：

- 初始状态
- 生成中状态
- 生成成功状态
- 生成失败状态
- 无结果状态

## 后续预设功能命名

后续增加提示词预设时，统一使用以下叫法：

- 预设提示词区
- 预设分类
- 预设项
- 预设说明

## 与后端绑定的字段名

只有接口字段继续保留英文，并且必须与当前后端契约一致：

- `prompt`
- `size`
- `quality`
- `outputFormat`
- `n`
- `background`
- `moderation`

## 使用原则

- 后续讨论优先使用本文档中的中文区域名
- 先说功能区域，再说具体改动
- 如果后续页面结构调整，先改本文档，再改代码
